The Myers-Briggs Personality Type Indicator is a self-report inventory designed to identify a person's personality type, strengths, and preferences.

This website is for those who are interested in getting to know the ENTP and the ISTP types of personalities. Anyone can know their personality types by taking this free test

ISTP is an acronym used to describe one of the sixteen personality types. It stands for Introverted, Sensing, Thinking, Perceiving.

ENTP is an acronym used to describe one of the sixteen personality types. It stands for Extraverted, iNtuitive, Thinking, Perceiving.
